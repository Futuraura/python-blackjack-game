## Requirements

- Python >=3.12
- Every python module in requirements.txt

## Installation

1. Clone or download the repository to your local machine

2. Navigate to the project directory and install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the game:

   ```bash
   python main.py
   ```
